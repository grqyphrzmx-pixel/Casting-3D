三、铸造专用约束规则
================================================================================

✓ 拔模斜度规则
  - 砂型铸造: 外表面0.5°-3°, 内表面1°-5°
  - 压铸: 外表面0.25°-2°, 内表面0.5°-2.5°
  - 熔模铸造: 一般不需要（0°-0.5°）
  - 按特征高度自动调整

✓ 圆角规则
  - 外圆角: R = (1/3 ~ 1/2) × 壁厚
  - 内圆角: R = (1/2 ~ 2/3) × 壁厚
  - 接头类型特殊处理（L/T/X/Y型）
  - 最小圆角半径1.5mm

✓ 壁厚规则
  - 铝合金砂型: 3-4mm
  - 铝合金压铸: 1-1.5mm
  - 灰铸铁: 4-6mm
  - 铸钢: 5-8mm
  - 壁厚变化率 ≤ 2:1

✓ 分型面规则
  - 位于最大轮廓处
  - 优先选择平面
  - 铝合金: 低分型
  - 铸铁/铸钢: 中间分型

✓ 浇注系统规则
  - 浇口从厚大部位引入
  - 冒口模数 ≥ 1.2 × 铸件模数
  - 流道比: 直:横:内 = 1.15:1.1:1
